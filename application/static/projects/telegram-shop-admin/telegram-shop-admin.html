Telegram store, working on webhooks, with payment systems, and connected to the admin panel, in which the management of users/goods and viewing of statistics takes place. In the screenshot you can see a test instance of the admin, running on the local area. Powered By:
  • aiogram
  • flask
  • redis
  • PostgreSQL
  • Docker
  • nginx
  • gunicorn
The bot has an inline keyboard as a source of interaction. In the admin panel is implemented RBAC, that is, worked out the roles and rights, which allows individual sellers to give their own offices, to download digital goods, and not to get mixed up with other sellers !
